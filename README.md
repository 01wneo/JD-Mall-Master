## 介绍

一款高仿京东商城的UI，基于MVP的Retrofit2(okhttp3)+rxjava+dagger2+greendao+glide。该项目系仿京东商城,属于独立开发者作品,仅供参考学习，拒绝做一切商业用途，如有侵权，请告知删除
